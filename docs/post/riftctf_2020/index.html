[Forensics]: 0x0003 (300 points)

Challenge

  1. identify the file format.
  1. read about the file format.
  2. see which properties this particular file has.
  3. and fix the file to get the flag.
  4. brute-forcing won’t help but you can do whatever you want.
  5. flag format ritsCTF{<—flag-here—>}. Good Luck!

  • flag.zip
  • readme.txt
