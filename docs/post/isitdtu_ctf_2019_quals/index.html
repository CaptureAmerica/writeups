Solution

解き方は見ての通りなんですが、アスキーアートを判別して数値に直して計算して結果を送るだけです。

とりあえず、何度かアクセスして、全部の数字と記号のアスキーアートを集めます。

スライスで7文字幅ずつチェックするようにしました。

ポイントとなるところは、数字の「1」と記号は横幅が狭いので、次の文字を見に行く際の移動幅が異なる事くらいです。

文字の判別ができてしまえば、後はeval()するだけです。
